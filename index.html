Conceptos de Programación

By Laura García (DevDygarcia)

Programa

Conjunto de instrucciones que una computadora sigue para realizar una tarea específica.

Tipos de programas:

  • Software de sistema: Programas que gestionan y controlan los recursos del hardware, como el sistema operativo.
  • Software de aplicación: Programas diseñados para realizar tareas específicas para el usuario, como procesadores de texto o navegadores web.

Código fuente

Conjunto de instrucciones escritas en un lenguaje de programación que pueden ser leídas y entendidas por los programadores.

Código objeto

Código binario generado por el compilador a partir del código fuente, que puede ser ejecutado por la computadora.

Palabras reservadas

Palabras que tienen un significado especial en un lenguaje de programación y no pueden ser utilizadas como identificadores (nombres de variables, funciones, etc.).

Comentarios

Notas en el código fuente que son ignoradas por el compilador o intérprete y sirven para explicar y documentar el código.

Concepto de librería

Conjunto de funciones y procedimientos predefinidos que pueden ser utilizados en programas para realizar tareas comunes.

Concepto de Método o función

Bloque de código que realiza una tarea específica y puede ser llamado desde otras partes del programa.

Método principal main

Punto de entrada de un programa en muchos lenguajes de programación, donde comienza la ejecución del código.

Conceptos de Datos

Información que puede ser procesada por un programa. Los datos pueden ser de diferentes tipos, como números, texto, imágenes, etc.

Variables

Espacios en la memoria de la computadora que almacenan datos y cuyo valor puede cambiar durante la ejecución del programa.

Constantes

Valores que no cambian durante la ejecución del programa y se definen una vez al inicio.

Tipos de datos

Categorías de datos que determinan qué tipo de valores pueden ser almacenados y cómo se pueden manipular. Ejemplos incluyen:

  • Numéricos: Datos que representan números.
  • Entero: Tipo de dato numérico que representa números enteros sin decimales.
